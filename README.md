# React Movie App
https://jojoyap.github.io/react-movie-app/#/

## Features

- Search movies
- Display recommended movies
- Favorite movies using LocalStorage
- Favorites page to view and remove saved movies
- Hover effects with animation (scale and shadow)
- Click on a movie to open a detailed modal
- Header with search bar and favorites link
- Responsive design for desktop and mobile

---

## Tech Stack

- React 18 + TypeScript + SCSS
- Vite
- GSAP
- Lottie
- react-icons
- react-router-dom
- TMDB API
